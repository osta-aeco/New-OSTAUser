# New-OSTAUser
Creates new AzureAD/Office365 users for OSTA-AECO.

```powershell
New-OSTAUser
	-FirstName <String[]>
	-LastName <String[]>
	-Title <String[]>
	-Department <String[GeneralAssembly|ExecutiveCouncil|BoardDirectors]>
	-LicenseType <String[Base|Enhanced]>
	[-OSTADomain <String[]>]
```

Check out the [Wiki](https://github.com/osta-aeco/New-OSTAUser/wiki) for this repository for information about the module, including installation instructions.
